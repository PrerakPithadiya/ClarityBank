# **App Name**: ClarityBank

## Core Features:

- Account Balance Display: Displays the current account balance in a prominent, easily readable format.
- Funds Deposit: Allows users to deposit funds into their account using a simple interface.
- Funds Withdrawal: Enables users to withdraw funds from their account, with appropriate security measures.
- Transaction History: Provides a detailed transaction history, filterable by date and type.

## Style Guidelines:

- Primary color: A professional, calming blue (#2962FF) evoking trust and security.
- Background color: Light, desaturated blue (#E8F0FE) to provide a clean, uncluttered backdrop.
- Accent color: Silver-gray (#B0BEC5) for subtle highlights and dividing elements, adding a touch of sophistication.
- Body and headline font: 'Inter' sans-serif for a modern and readable style.
- Use simple, professional icons to represent transaction types (deposits, withdrawals, etc.).
- Employ a clean, minimalist layout with ample white space to enhance readability and focus.
- Subtle animations for transactions.
